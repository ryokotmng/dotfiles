# Ryoko's dotfiles
## Installation
### Using Git
```
git clone https://github.com/ryokotmng/dotfiles.git
```
## Install Plugins
##### dein
under ~/dotfiles
```
sh ./installer.sh ~/.cache/dein
```
Open vim and install dein
:call dein#install()
##### fugitive
under ~/dotfiles/.vim/bundler
```
git clone https://github.com/tpope/vim-fugitive.git
```
